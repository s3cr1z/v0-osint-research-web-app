# Project: OSINT Investigation Dashboard

## Overview
This project implements a pixel-perfect OSINT investigation dashboard 
using Next.js 15, Tailwind CSS v4, shadcn/ui, and Rete.js.

## Key Implementation Files
- Read `/docs/OSINT_DASHBOARD_SPEC.md` for complete specifications
- Follow all color values, spacing, and component structures exactly

## Technology Stack
- Next.js 15 with App Router
- Tailwind CSS v4
- shadcn/ui components
- Rete.js for node-based canvas
- Recharts and D3 for visualizations

## Critical Rules
1. Use exact hex color values from specification
2. Maintain consistent 4px spacing system
3. Follow component structure precisely
4. Implement all Rete.js nodes as documented.
